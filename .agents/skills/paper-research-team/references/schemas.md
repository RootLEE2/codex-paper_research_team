# Paper Research Schemas

This file is an index. Load only the schema file needed for the current phase.

## Session Outputs

Use `references/schemas/session-files.md` for:

- default session directory
- always-required session files
- conditional synthesis/review files
- candidate/evidence/screening/selected/pending-PDF file shapes

## Pending Records

Use `references/schemas/pending-record.md` only when a selected paper lacks a PDF, preprint, or confirmed full text.

Pending records go under:

`papers/obsidian/pending-notes/`

They must not be placed under:

`papers/obsidian/notes/`

## Paper Notes

Use `references/schemas/paper-note.md` when writing reusable notes under:

`papers/obsidian/notes/`

## Note Validation

Use `references/schemas/note-validation.md` when checking generated or materially updated paper notes against source PDFs/full text before synthesis.

## Blocked Outputs

Use `references/schemas/blocked-outputs.md` only when required selected papers still lack PDFs, preprints, or confirmed full text.

Blocked outputs are placeholders for:

- `synthesis.md`
- `professor-review.md`

They must not look like final synthesis or final professor review.
