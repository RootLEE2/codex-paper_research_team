# Source Priority

Use this when planning search strategy or updating source dependencies.

Default order:

1. OpenAlex
2. Semantic Scholar through paper-search MCP
3. Zotero library reuse after seed discovery
4. Consensus MCP when question-driven peer-reviewed validation adds value

## OpenAlex

Primary source for metadata, venues, citations, references, related works, author metadata, deduplication, and broad candidate search.

## Semantic Scholar

Secondary source for relevance search, abstracts, PDF URL discovery, and cross-checking. Use focused result limits.

## Zotero

Use after seed discovery for deduplication, existing-PDF lookup, prior Obsidian note reuse, and recovery of genuinely core or seminal papers already in the library.

Do not start from Zotero unless the user explicitly asks to analyze the existing library.

## Consensus

Use for question-driven peer-reviewed search, structured validation, and filters when it adds value beyond OpenAlex.
