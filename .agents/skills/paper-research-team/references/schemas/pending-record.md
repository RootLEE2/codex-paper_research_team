# Pending Obsidian Record Schema

Use this only for selected papers without a PDF, preprint, or confirmed full text.

Keep records concise. The folder should contain only papers that still require user-provided full text.

Store under:

`papers/obsidian/pending-notes/`

Do not store under:

`papers/obsidian/notes/`

```markdown
---
title: "<paper title>"
year: 2026
doi: "10.xxxx/yyyy"
pdf_status: "pdf_missing"
status: "pending_full_text"
conceptual_status: "core"
---

# <paper title>

## Pending Reason

## Metadata Available

## Why Needed

## Resume Instructions
```
